# SOC Basics Lab

## Objective
To understand basic Security Operations Center (SOC) concepts including alert monitoring, log analysis, and incident documentation.

## Activities
- Reviewed simulated security alerts
- Identified suspicious login patterns
- Practiced documenting incidents using standard incident fields

## Tools Used
- Splunk (lab environment)
- Sample log files

## Key Takeaways
- Importance of monitoring and alert prioritization
- Clear documentation helps effective incident response
- SOC analysts must balance speed and accuracy
