# Sample Incident Report

**Incident Type:** Suspicious Login Activity  
**Severity:** Medium  
**Detection Method:** SIEM Alert  
**Description:** Multiple failed login attempts detected from a single IP address within a short time frame.

## Initial Analysis
- Possible brute-force attempt
- No successful login observed

## Action Taken
- Alert reviewed and documented
- IP flagged for monitoring
- Incident escalated for further review

## Lessons Learned
- Importance of login monitoring
- Early detection reduces risk
